# Sales Data Analysis & Visualization

## Description
This project generates synthetic sales data and performs data analysis
and visualization using Python.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Features
- Synthetic data generation
- Product-wise and region-wise sales analysis
- Data visualization using bar, pie, and line charts

## How to Run
```bash
pip install numpy pandas matplotlib
python sales_analysis.py
